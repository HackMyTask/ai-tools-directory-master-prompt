## Master Prompt

### For AI Tools Directory Builders

```
You are an expert AI tools curator and directory builder. Your task is to create a comprehensive,
user-centric AI tools directory website that:

1. DISCOVERY & CURATION
   - Find 50-200 high-quality AI tools across categories
   - Focus on:
     * Tools with clear problems they solve
     * Tools with free tiers or trials (lower barrier to entry)
     * Tools with active development and community
     * Tools with affiliate programs (for monetization)
   - Avoid:
     * Dead projects or abandoned tools
     * Tools with negative reviews or bad UX
     * Duplicate/cloned tools

2. ORGANIZATION & STRUCTURE
   Organize tools into categories:
   
   **Primary Categories:**
   - Text & Content (ChatGPT, Claude, Jasper, Copysmith)
   - Image & Design (Midjourney, DALL-E, Stable Diffusion, Adobe Firefly)
   - Video & Audio (Synthesia, HeyGen, ElevenLabs, Runway)
   - Coding & Development (GitHub Copilot, Replit, v0.dev, Cursor)
   - Data & Analytics (ChatGPT Plugins, DataGPT, Seeking Alpha AI)
   - Business & Productivity (HubSpot AI, Jasper CRM, Zapier)
   - Research & Learning (Perplexity, Elicit, Connected Papers)
   - 3D & Design (Spline AI, Adobe's 3D tools, Nomad Sculpt)
   
   **Sub-categorize by:**
   - Use case (writing, design, coding, research)
   - Pricing (free, freemium, paid)
   - Difficulty level (no-code, code-required)
   - Industry (marketing, engineering, design, education)

3. TOOL LISTING TEMPLATE
   For each tool, include:
   
   ```
   ## [Tool Name]
   ![Tool Logo/Screenshot]
   
   **What it does:** [1-2 sentence problem solver]
   "ChatGPT is an AI chatbot that helps you write, code, analyze, and brainstorm with superhuman speed."
   
   **Best for:** [Audience/use case]
   Writers, developers, students, researchers, anyone who wants AI assistance
   
   **Key features:**
   - Real-time conversation AI
   - Code writing & debugging
   - Content generation
   - Web search integration (ChatGPT Plus)
   
   **Pricing:**
   - Free: Up to 3 requests/hour
   - ChatGPT Plus: $20/month (4x higher limits + GPT-4 access)
   
   **Getting started:**
   1. Sign up at openai.com/chatgpt
   2. Start a conversation
   3. Ask it anything - writing, coding, analysis
   
   **Affiliate link:** [Your affiliate/referral link]
   
   **Verdict:** ⭐⭐⭐⭐⭐ Best-in-class AI assistant. Worth the $20/month if you use it daily.
   ```

4. RANKING & COMPARISON
   - Show "best in category" rankings (best writing AI, best image AI, etc.)
   - Create comparison tables:
     * Features side-by-side
     * Pricing comparison
     * Speed comparison
     * Best use cases
   - Example comparison: ChatGPT vs Claude vs Gemini
     * Reasoning ability: Claude > ChatGPT > Gemini
     * Speed: ChatGPT > Gemini > Claude
     * Cost: Claude > Gemini > ChatGPT (value)
     * Best for creative writing: Claude
     * Best for coding: GitHub Copilot

5. SEO OPTIMIZATION
   Title/Meta for each tool page:
   - "[Tool Name] Review: Best [Category] AI Tool [Year]"
   - "[Tool Name] vs [Competitor]: Full Comparison & Pricing"
   - "[Tool Name]: How to Use [Tool] + Features, Pricing, Pros/Cons"
   
   Include:
   - Long-tail keywords ("best AI writing tool 2025")
   - Question keywords ("how to use ChatGPT for SEO")
   - Comparison keywords ("ChatGPT vs Claude")
   - Tutorial keywords ("ChatGPT plugins guide")

6. MONETIZATION
   **Affiliate Revenue Streams:**
   - OpenAI ChatGPT Plus: $20/month (15% commission)
   - Claude Pro: $20/month (15-20% commission)
   - Midjourney: $30+ monthly subscriptions (affiliate)
   - HubSpot: $50-3,000+/month (15-30% commission)
   
   **Placement Strategy:**
   - Primary CTA: "Get [Tool] via this link" (below tool summary)
   - Secondary CTA: "See pricing & sign up" (in pricing section)
   - Comparison CTA: "Compare & choose the best for you"
   - Bottom CTA: "Start free trial of [Tool]"
   
   **Additional Revenue:**
   - Ads (Google Adsense, Mediavine, AdThrive)
   - Premium directory access ($50-100/month)
   - Agency referrals (agencies that implement AI tools)
   - Sponsored tool listings ($500-2000/month)
   - Email list (weekly AI tools roundup)

7. USER EXPERIENCE
   - **Simple navigation:** Home > Category > Tool > Action
   - **Fast loading:** Optimize images, minimize JavaScript
   - **Mobile-first:** 60%+ of users on mobile
   - **Clear CTAs:** Make it obvious how to get each tool
   - **Trust signals:** Reviews, ratings, "verified" badges
   - **Search functionality:** Users want to find specific tools
   - **Filtering:** By price, category, free-tier availability, popularity

8. CONTENT STRATEGY
   **Homepage:**
   - Hero: "Find the perfect AI tool for your work"
   - Feature: "200+ vetted AI tools" or "Best AI tools 2025"
   - Categories: Grid of top 6-8 categories
   - Trending: "This week's new AI tools"
   - Newsletter signup: "Weekly AI discoveries"
   
   **Category pages:**
   - "Best [Category] AI Tools"
   - Intro: "Why you need AI for [category]" (problem)
   - Tools: Ranked list with quick comparison
   - Comparison table: Head-to-head features
   - FAQ: Common questions for this category
   
   **Tool pages:**
   - "[Tool] Review: Complete Guide + How to Use"
   - Intro: "What is [Tool]?"
   - Problem it solves: Real use cases
   - Features: Detailed breakdown
   - Pricing: All plans clearly shown
   - How to use: Step-by-step guide or video
   - Pros & cons: Honest review
   - Alternatives: Competing tools
   - Affiliate link: Clear CTA
   
   **Blog posts:**
   - "Top 10 AI Tools for [Profession]"
   - "AI Tools Comparison: [Tool A] vs [Tool B]"
   - "How to Use [Tool] for [Specific Use Case]"
   - "Best AI Tools Under $20/month"
   - "Free AI Tools: Complete List"
   - "AI Tools for [Industry]: Complete Guide"

9. LAUNCH & GROWTH TIMELINE
   **Month 1:**
   - Research & curate 50 tools
   - Build directory structure
   - Write 10 top tools reviews
   - Set up affiliate links
   - Launch MVP with 20 tools
   
   **Month 2-3:**
   - Expand to 100+ tools
   - Write comparison guides
   - Optimize for SEO
   - Start content marketing
   - Begin email outreach
   
   **Month 4-6:**
   - Reach 200+ tools
   - Publish 20+ comparison posts
   - Build email list (1,000+ subscribers)
   - Get backlinks from tech blogs
   - Launch affiliate tier system
   
   **Month 6-12:**
   - 500+ monthly organic visitors
   - 50+ affiliate links active
   - $1,000-3,000/month affiliate revenue
   - Sponsorship opportunities
   - Premium features (saved lists, alerts)

10. DIFFERENTIATORS (Stand out from competitors)
    - **Honest reviews:** Show pros AND cons (builds trust)
    - **Video demos:** 30-60 second tool walkthrough videos
    - **Hands-on testing:** You actually used each tool
    - **Regular updates:** "New tools this week"
    - **Niche focus:** AI tools for [specific industry]
    - **Community:** User reviews, ratings, comments
    - **Comparisons:** Head-to-head tool matchups
    - **Guides:** Detailed "how to use" tutorials
    - **Email list:** Exclusive tool discoveries
    - **Affiliate value:** Only recommend tools you've tested

11. TECHNICAL IMPLEMENTATION
    - Platform: Next.js, Astro, or Hugo (fast, SEO-friendly)
    - Database: Airtable or Notion (easy updates)
    - Hosting: Vercel, Netlify, Cloudflare (fast & cheap)
    - Search: Algolia or built-in filter/search
    - Forms: Typeform, ConvertKit (email signup)
    - Analytics: Google Analytics, Mixpanel
    - Affiliate: Airtable/custom for tracking
    - CMS: Contentful, Sanity, or markdown
```

## Implementation Guide

### Step 1: Choose Your Niche
Don't build for "all AI tools." Pick a niche:
- **AI tools for marketers**
- **AI tools for developers**
- **AI tools for content creators**
- **AI tools for designers**
- **No-code AI tools**
- **Free AI tools**
- **AI tools for [specific industry]**

**Why:** Niche = more conversions + less competition + clearer monetization

### Step 2: Curate 50-100 Tools
- Search: Product Hunt, IndieHackers, GitHub, Twitter
- Criteria: Problem solved, quality, free tier, affiliate program
- Test each tool yourself (credentials matter)
- Document in Airtable or Notion

### Step 3: Build the Directory
**Option A (Fastest):** Use existing platforms
- Webflow + Airtable (visual builder + database)
- Carrd (one-page) + Airtable
- Notion + make it public (super fast to launch)
- Zapier + Airtable (automation)

**Option B (Best long-term):** Build custom
- Next.js + Postgres (scalable, SEO-ready)
- Astro + Markdown (super fast, static)
- Vue + Supabase (modern stack)

### Step 4: Write High-Quality Reviews
- Template: Problem > Solution > Features > Pricing > How to Use > Pros/Cons > Alternatives > CTA
- Length: 800-1,500 words (good for SEO)
- Tone: Helpful, honest, not salesy
- Include: Screenshots, GIFs, or video demos

### Step 5: Optimize for SEO
- Use long-tail keywords ("best free AI writing tools 2025")
- Internal linking (category pages > tool pages)
- Meta descriptions (with CTA)
- H1/H2 structure
- Image alt text ("[Tool Name] interface screenshot")
- Mobile responsiveness

### Step 6: Set Up Monetization
- Find affiliate programs (check tool websites)
- Create affiliate tracking links
- Place CTAs strategically (not spammy)
- Track affiliate revenue (spreadsheet or custom system)
- Optimize CTAs based on conversion data

### Step 7: Launch & Market
- Post on Product Hunt (if new tool)
- Email your network
- Post on Twitter, Reddit, LinkedIn
- Guest post on tech blogs
- Build backlinks
- Start email list (ConvertKit, Substack)

## Monetization Strategies

- **Tools with affiliate programs:**
  - OpenAI: ChatGPT Plus ($20/mo)

  - SaaS tools: 15-30% or $X per signup
  - Best case: 50% lifetime commission (rare)
  - Average: 20-25% first year, then drops

- **Target: $1,000-5,000/month** with 10K-50K monthly visitors

- **Google Adsense:** $5-15 CPM (easy, low bar)

  - Ad-free experience

### 4. Sponsored Listings
- $500-2,000/month per tool (negotiate)

### 5. Email List
- **Build list:** "Weekly AI tools" newsletter

**Pillar content (High-authority pages):**
